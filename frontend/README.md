# MomentsShare 小程序前端

基于 Vue 3 + Vite 构建的朋友圈分享平台前端，采用手机模拟器设计，提供沉浸式移动端体验。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器 (端口 3000)
npm run dev

# 构建生产版本
npm run build
```

## 📱 特色功能

- **手机模拟器**: 在浏览器中模拟 iPhone 14 Pro 外观
- **完整页面**: 登录/注册、首页 Feed、发布、发现、消息、个人中心
- **AI 集成**: 文案润色、智能标签推荐
- **社交功能**: 好友系统、评论互动

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **构建**: Vite 5
- **路由**: Vue Router 4
- **状态**: Pinia
- **HTTP**: Axios
- **样式**: SCSS
